# SupplyChain前端

## 前端：
+ 语言: Vue 2.x
+ 框架: @vue/cli 4.5.11
+ 界面库: ant-design-vue
+ 网络请求组件: nodejs-axios

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
